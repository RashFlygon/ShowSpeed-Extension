ShowSpeed

Privacy Policy

Effective date: April 5, 2026

ShowSpeed is a Chrome extension for Pokemon Showdown that adds Speed tooltips and Speed creep tools to the teambuilder.

What Data The Extension Accesses

ShowSpeed accesses website content on the following domains:

  • https://play.pokemonshowdown.com/
  • https://smogtours.psim.us/

This includes teambuilder page content needed for the extension's features, such as displayed Pokemon, stats, EVs, IVs, items, and values entered by the user in the teambuilder.

How Data Is Used

ShowSpeed uses this information only to provide its user-facing features, including:

  • Speed hover tooltips
  • Speed tier calculations
  • Speed creep calculations
  • Saved extension settings and recent speed creep searches

How Data Is Stored

ShowSpeed stores user preferences locally using Chrome extension storage so settings persist between browser sessions.

ShowSpeed may also store recent speed creep search values locally in browser storage on the user's device to preserve the last-used target and related inputs.

How Data Is Shared

ShowSpeed does not sell, transfer, or share user data with third parties.

ShowSpeed does not use user data for advertising, profiling, creditworthiness, lending decisions, or any purpose unrelated to the extension's single purpose.

Remote Code And External Resources

ShowSpeed does not use remote code. All executable code is packaged with the extension.

The extension popup may load a remote animated image from Pokemon Showdown for visual display only. This image is not executable code and is not used to collect or transmit extension data.

Permissions

ShowSpeed uses Chrome's storage permission to save extension settings locally. It runs only on the supported Pokemon Showdown domains listed above.

Children's Privacy

ShowSpeed is not directed to children and does not knowingly collect personally identifiable information.

Changes To This Policy

This privacy policy may be updated from time to time. Any updates will be posted on this page with a revised effective date.
